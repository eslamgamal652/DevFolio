# DevFolio

DevFolio is a personal portfolio website designed to showcase a developer's profile, skills, services, projects, and contact information.

The project is built using HTML, CSS, and Bootstrap, with a clean and responsive design suitable for personal branding and professional presentation.

## Features

- Modern portfolio landing page
- About section
- Skills section
- Services section
- Portfolio/projects section
- Statistics section
- Testimonials section
- Blog section
- Contact section
- Responsive design for desktop, tablet, and mobile screens

## Technologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript
- Font Awesome

## How to Run the Project

To run this project locally:

### 1. Clone the repository

```bash
git clone https://github.com/eslamgamal652/DevFolio.git
```

### 2. Open the project folder

```bash
cd DevFolio
```

### 3. Open the project

Open the `index.html` file in your browser.

Or you can use Live Server in Visual Studio Code.

## Website Sections

### Home

The main landing section that introduces the portfolio owner.

### About

A section that includes personal information, skills, and a short biography.

### Services

Displays the services offered such as web design, web development, and responsive design.

### Portfolio

Shows previous work and projects.

### Blog

Contains articles or posts related to development and technology.

### Contact

Allows visitors to get in touch through contact information or a contact form.
